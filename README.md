#乐兴REACT-NATIVE android示例项目
----

## 使用说明

首先clone项目repo `git clone https://github.com/LexingSoftware/RNApp.git`

然后 `cd RNApp && npm i`

依赖安装完成后 如果是windows系统 需要打开新的CMD 执行 `react-native start` 等待启动完成后,不要关闭 在原先的CMD窗口输入`react-native run-android`即可
